# Aswedaul Core Service

Backend service for the **Aswedaul_Ed Platform**, responsible for managing users, academic activities, admissions, content, and internal school operations.  
This service exposes both **REST APIs** and **GraphQL APIs** for client applications (web and mobile).

Features

-User Management
- User registration and onboarding
- Student, teacher, and admin management
- Role-based access control (RBAC)
- Permissions management
- User profiles and status handling

- Authentication & Authorization
- JWT-based authentication
- Role and permission guards
- Token validation and refresh
- Secure access to REST and GraphQL endpoints

- Admissions Management
- Student admission applications
- Admission status tracking
- Grade and class assignment
- Admission workflow management

 Activities Management
- Academic and non-academic activities
- Activity listing CRUD routes

- Leave Requests
- Student and staff leave requests
- Approval/rejection workflow
- Leave history tracking

- Content Management
- FAQs management
- Galleries (images and media)
